# THE FLI IELTS Training

A premium, highly professional, ultra-clean EdTech landing page for "THE FLI IELTS Training," a specialized test-prep division of The Polyglot Academy.

## Project Structure
- `index.html`: Base file with Tailwind CSS config, font imports, and CDN script tags.
- `app.js`: Main React entry point.
- `components/`: Contains modular UI elements: Navbar, Hero, Features, Courses, Results, CTA, Contact, Footer.

## Tech Stack
- React 18 (CDN)
- Tailwind CSS (CDN)
- Lucide Icons (Static Font CDN)
- Babel Standalone (CDN)

## Design Guidelines
- **Colors:** Premium Black (#111111), Primary Yellow (#F5C400), White, Dark Gray, Light Gray.
- **Typography:** Poppins (Headings), Inter (Body text).
- **Style:** Minimalist, Luxury Educational, UI/UX focused. Avoid clutter. Emphasize spacing and typography.
